# Posetrip

Posetrip is a high-performance, intuitive auto-clicker designed for seamless operation. Built with advanced multi-threading, it ensures extremely fast and stable clicking without system performance drops.

## Key Features
* **Advanced Timing:** Supports multiple modes including QPC busy-wait, hybrid timers, and tick-aligned synchronization for precise performance.
* **Click Customization:** Adjustable CPS with randomization, double-click support, drop chance, and burst boost modes.
* **Stealth & Security:** Features anti-capture protection (OBS/Discord), random title generation, and a self-destruct function.
* **UI:** Compact, RGB-accented UI, animated tabs, and customizable keybinds.
* **Targeting:** Intelligent slot whitelisting, inventory filtering, and aim jitter options.
